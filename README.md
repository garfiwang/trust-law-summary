# 信託法總整理 互動教學簡報

本專案為「問大師家族辦公室」品牌風格之信託法規（第一條至第六十八條）互動式教學簡報，完全基於結構化 Markdown 檔案生成。

## 專案特色

- **Reveal.js 互動簡報**：支援鍵盤方向鍵、滑鼠點擊切換投影片，並支援 overview 網格概覽（按 `ESC` 鍵）與搜尋功能（按 `CTRL + F` 或 `CMD + F` 進行法條檢索）。
- **單頁無捲軸設計**：所有法規文字已完成排版優化，完全呈現在單一畫面上，無需讀者上下滾動。
- **品牌識別色與標語**：
  - **背景**：問大師標準米白底色 (`#F9F5F0`)
  - **文字**：深褐主色 (`#3E2115`)
  - **強調**：深金輔色 (`#C9A227`)
  - **標語**：「信託不會改變你的生活，信託可以確保未來你的生活不會被改變」
